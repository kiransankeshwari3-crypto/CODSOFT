# Iris Flower Classification

## Project Overview
This project builds a Data science model to classify Iris flowers into
Setosa, Versicolor, and Virginica based on sepal and petal measurements.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
- Loaded Iris dataset
- Data visualization
- Feature scaling
- Model training using Logistic Regression
- Model evaluation

## Result
The model achieves high accuracy in predicting Iris flower species.
